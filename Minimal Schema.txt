
PODS contains a LIST OF PODS
	Each POD contains a LIST OF SENSORS
	Each POD contains a CURRENT DATASET
	Each POD contains a CURRENT GATEWAY
	Each POD contains LOCATION AND DEPLOYMENT METADATA

DATASETS
	Each DATASET contains a LIST OF USERS
	Each DATASET contains a LIST OF DATA RECORDS
		Each DATA RECORD contains a POD, SENSOR, VALUE, AND TIMESTAMP

GATEWAYS contains a LIST OF GATEWAYS
	Each GATEWAY

SENSORS contains a LIST OF SENSORS
	Each SENSOR
	
USERS contains a LIST OF USERS
	Each USER
